REPRYNTT — first-time setup

Welcome.

Pick how you want to run Repryntt. You can switch later from Settings.

Cloud — easiest

Sign in with repryntt.com

Use a hosted Repryntt account. Your agent runs on the cloud or on a paired machine you set up earlier. No Python, no terminal.
  • Works on any computer
  • Just need an API key from repryntt.com
  • ~10 seconds to set up
Local — for advanced users

Connect to a local install

You've already installed the Repryntt daemon on this computer (via install.py). The desktop app will connect to localhost:8089.
  • Fully offline-capable
  • Requires the daemon to be running
  • Bring your own LLM keys
CLOUD — sign in

Sign in to repryntt.com

Paste your API key from repryntt.com/dashboard/api-keys. It's a long string starting with rkey_.

Stored locally on this machine only. Never sent anywhere except api.repryntt.com.
LOCAL — connect

Connect to your local daemon

Repryntt Desktop will look for a daemon running at the address below.

Default is http://localhost:8089. Change only if you moved the daemon.

Daemon not running? Open a terminal and run repryntt start, then come back and click Test connection. The full install guide is at github.com/ai158z/repryntt-public.
